# 🍳 智能厨房助手

基于智谱AI大模型的智能厨房助手，提供食材识别、食谱推荐、营养分析和语音交互等功能。

## 功能特性

- **📷 食材识别**: 使用GLM-4V模型智能识别图片中的食材
- **🍽️ 食谱推荐**: 基于可用食材推荐个性化食谱
- **📊 营养分析**: 分析食材营养成分和热量
- **🎤 语音助手**: 语音识别和语音播报功能
- **🔍 智能推荐**: 使用嵌入向量进行相似度匹配

## 项目结构

## 快速开始

### 1. 环境准备

确保您已安装Python 3.8+，然后克隆或下载本项目。

### 2. 安装依赖

```bash
cd kitchen_assistant_zhipu
pip install -r requirements.txt
```

### 3. 配置API密钥

1. 获取智谱AI API密钥：访问 [智谱AI开放平台](https://open.bigmodel.cn/)
2. 编辑 `.env` 文件，将您的API密钥填入：

```env
ZHIPU_API_KEY=your_actual_api_key_here
```

### 4. 启动应用

```bash
streamlit run app.py
```

应用将在本地启动，默认地址为：http://localhost:8501

## 详细使用说明

### 食材识别功能

1. 点击"📷 食材识别"标签
2. 上传包含食材的图片（支持PNG、JPG、JPEG格式）
3. 点击"开始识别食材"按钮
4. 系统将识别食材并提供：
   - 食材清单和数量估计
   - 新鲜度评估
   - 烹饪建议
   - 食品安全分析
   - 热量估算

### 食谱推荐功能

1. 点击"🍽️ 食谱推荐"标签
2. 输入可用食材（用逗号分隔）
3. 设置偏好（难度、时间等）
4. 点击"推荐食谱"按钮
5. 系统将提供：
   - 个性化食谱推荐
   - 详细烹饪步骤
   - 语音播报指导
   - 定制食谱生成

### 营养分析功能

1. 点击"📊 营养分析"标签
2. 输入食材清单
3. 选择份量大小
4. 点击"分析营养"按钮
5. 系统将提供：
   - 营养成分分析
   - 热量计算
   - 健康益处
   - 饮食建议
   - 7天饮食计划

### 语音助手功能

1. 在侧边栏启用语音功能
2. 点击"🎤 语音助手"标签
3. 使用语音输入：
   - 点击"开始录音"进行语音识别
   - 系统将分析厨房相关命令
   - 提供语音回复
4. 语音播报：
   - 输入文本进行语音播报
   - 快速指令（食材介绍、烹饪指导）

## API接口说明

### 智谱AI模型使用

- **GLM-4**: 通用对话和文本生成
- **GLM-4V**: 图像识别和分析
- **GLM-ASR-2512**: 语音识别
- **Embedding-2**: 文本嵌入向量
- **GLM-TTS**: 文本转语音

### 主要模块功能

#### ZhipuClient
- 统一的API客户端封装
- 支持所有智谱AI服务
- 错误处理和重试机制

#### ImageRecognition
- 食材识别和分类
- 食品安全分析
- 热量估算

#### RecipeRecommender
- 基于嵌入向量的相似度匹配
- 个性化食谱推荐
- 定制食谱生成

#### NutritionAnalyzer
- 营养成分分析
- 饮食计划制定
- 食物搭配建议

#### VoiceAssistant
- 语音录制和识别
- 命令意图分析
- 持续监听功能

#### TTSEngine
- 文本转语音
- 音频播放控制
- 厨房专用语音回复


## 故障排除

### 常见问题

1. **API密钥错误**
   - 检查 `.env` 文件中的ZHIPU_API_KEY是否正确
   - 确认API密钥有足够的额度

2. **依赖安装失败**
   - 使用Python 3.8+版本
   - 尝试使用虚拟环境
   - 检查网络连接

3. **语音功能问题**
   - 确保麦克风权限已开启
   - 检查音频设备驱动
   - 在安静环境下录音

4. **图片识别效果不佳**
   - 使用清晰、光线良好的图片
   - 确保食材在图片中明显可见
   - 避免复杂背景

### 性能优化建议

- 使用高速网络连接
- 限制同时处理的图片大小
- 适当调整语音录制时长
- 定期清理临时文件
